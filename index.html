Time-lapse

Durante el Tercer Evento de Software Libre y Economía Social que realizamos en el Centro Cultural de la Cooperación (Av. Corrientes 1543, Ciudad de Buenos Aires), llevamos adelante un hackaton con la idea de ayudar a hacer visibles los "números del cooperativismo" en el país. Este video representa en formato time-lapse toda la mañana en la que estuvimos trabajando para este objetivo.

Repositorio

Para obtener el repositio completo del código completo puedes ejecutar el siguiente comando: 

git clone https://github.com/gcoop-libre/CoopViz.git

O bien, visitar el repositorio desde la web.

Proyectos

En la jornada logramos materializar varios proyectos, esta es una lista de los resultados:

Visualización del Nacimiento de Cooperativas
» Ver reseña completa
» Ver video

La visualización nos ha permitido observar el nacimiento de cooperativas y federaciones entre los años 1927 y 2008.

El sitio web de osiux tiene una reseña super completa sobre cómo se realizó la visualización usando (Gourse).

Ubicación Geográfica de las cooperativas
» Ver demostración interactiva

El mapa permite visualizar la locación geográfica de varias cooperativas, pero también brinda detalles de cada una cuando el usuario hace click en alguno de los puntos.

Para crear el mapa se ha utilizado el sistema CartoDB y la información que genera el script de locación geográfica.

Presentacion linea de tiempo de Creación de cooperativas
» Ver demostración interactiva

Esta linea de tiempo permite ver secuencialmente la fundación de cooperativas y los datos asociados a las mismas

Los datos fueron procesados con la herramienta Google Refine

Grafo Interactivo de Federaciones y Cooperativas
» Ver demostración interactiva

Este ejemplo permita navegar las cooperativas de Facttic y Fecootra, viendo las relaciones entre ellas

Para la visualización se utilizó la biblioteca de visualizaciones InfoViz Toolkit

Linea de tiempo de Creación de cooperativas

Muestra la creación de las primeras ~2000 cooperativas y federaciones, cubriendo aproximadamente desde el año 1929 hasta 1970.

Para armarla se utilizó la biblioteca SIMILE Widgets y para preparar los datos se utilizó Google Refine.

Realizado durante el evento sles-2012 por miembros FACTTIC.
